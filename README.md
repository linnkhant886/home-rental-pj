# Home Rental Platform

A full-stack rental booking platform with secure payments and authentication.

## Tech Stack
- Next.js
- TypeScript
- PostgreSQL
- Prisma
- Stripe API
- Clerk Auth

## Features
- User authentication & role management
- Property listing & booking
- Secure Stripe checkout
- Booking history dashboard
- Form validation with Zod

## Deployment
Deployed on Vercel with optimized production build.
